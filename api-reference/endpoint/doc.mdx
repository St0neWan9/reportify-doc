## 文档内容

**URL**: `/v1/docs/{doc_id}`  
**方法**: `GET`  
**描述**: 获取指定文档的详细内容，包括文档信息及内容块（chunks）。

### 请求参数

| 参数名 | 类型 | 描述          |
|--------|------|---------------|
| doc_id | str  | 文档唯一 ID。 |

### 响应参数
| 参数名              | 类型    | 描述                                                           |
|------------------|-------|--------------------------------------------------------------|
| doc              | dict  | 文档信息，包含以下字段：                                                 |
| &nbsp;&nbsp;id             | str   | 文档唯一 ID。                                                     |
| &nbsp;&nbsp;title          | str   | 文档标题。                                                        |
| &nbsp;&nbsp;url            | str   | 文档网页链接。                                                      |
| &nbsp;&nbsp;file_url       | str   | 文档文件链接。                                                      |
| &nbsp;&nbsp;media_url      | str   | 媒体文件链接。                                                      |
| &nbsp;&nbsp;institution    | str   | 发布机构。                                                        |
| &nbsp;&nbsp;author         | str   | 作者。                                                          |
| &nbsp;&nbsp;published_date | str   | 发布时间，格式为 `YYYY-MM-DD`。                                       |
| &nbsp;&nbsp;category       | str   | 文档类别（`filings`, `transcripts`, `reports`, `news`, `files`）。 |
| &nbsp;&nbsp;market         | str   | 市场区域（`cn`, `hk`, `us`）。                                      |
| &nbsp;&nbsp;symbol         | str   | 股票代码。                                                        |
| &nbsp;&nbsp;company_name   | str   | 公司名称。                                                        |
| &nbsp;&nbsp;logo           | str   | 公司 Logo 链接。                                                  |
| &nbsp;&nbsp;summary        | str   | 文档摘要。                                                        |
| &nbsp;&nbsp;chunks         | array | 内容块数组，每个块包含以下字段：                                             |
| &nbsp;&nbsp;&nbsp;&nbsp;type         | str   | 内容类型，可能的值为 `text`，`table`，`image`，`formula`。                 |
| &nbsp;&nbsp;&nbsp;&nbsp;media_url    | str   | 内容块的媒体文件链接（如果类型为 `table`，`image`，`formula` 时提供图片链接）。         |
| &nbsp;&nbsp;&nbsp;&nbsp;content      | str   | 内容原始文本（适用于 `text` 或 `table` 类型）。                             |
| &nbsp;&nbsp;&nbsp;&nbsp;summary      | str   | 解析内容，适用于 `table`，`image`，`formula` 类型。                       |
| &nbsp;&nbsp;&nbsp;&nbsp;tags         | dict  | 标签字典，包含以下字段：                                                 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;key        | str   | 标签类型（如 `industry`, `company`, `person`, `event`）。            |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;value      | array | 标签内容数组。                                                      |
| &nbsp;&nbsp;&nbsp;&nbsp;metadatas    | dict  | 元数据字典，包含以下字段：                                                |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;key        | str   | 元数据类型（如 `period`, `entities`）。                               |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;value      | array | 元数据内容数组。                                                     |

### 请求示例

**cURL**
```bash
curl -X GET https://api.reportify.cn/v1/docs/1046895208953942016 \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json"
```

**Python 示例**
```python
import requests

url = "https://api.reportify.cn/v1/docs/1046895208953942016"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```
