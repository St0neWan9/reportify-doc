## 关注账号

**URL**: `/v1/channels/{channel_id}/follow`  
**方法**: `POST`  
**描述**: 关注账号

### 请求参数
| 参数名         | 类型   | 描述                                                      |
|----------------|--------|-----------------------------------------------------------|
| channel_id          | str    | 账号唯一 ID。                                              |

### 响应参数
| 参数名            | 类型    | 描述                                                           |
|----------------|-------|--------------------------------------------------------------|
| id           | str   | 账号唯一 ID。                                                     |
| name        | str   | 账号名称。                                                        |
| avatar_url   | str   | 账号头像。                                                        |
| description  | str   | 账号描述。                                                        |
| following  | bool   | 账号关注状态。                                                     |
