## 文档内容

**URL**: `/v1/docs/{doc_id}/raw_content`  
**方法**: `GET`  
**描述**: 获取指定文档的原始内容。

### 请求参数

| 参数名 | 类型 | 描述          |
|--------|------|---------------|
| doc_id | str  | 文档唯一 ID。 |

### 响应参数
| 参数名              | 类型    | 描述                                                           |
|------------------|-------|--------------------------------------------------------------|
| doc_id             | str   | 文档唯一 ID。                                                     |
| url            | str   | 文档网页链接。                                                      |
| file_url       | str   | 文档文件链接。                                                      |
| media_url      | str   | 媒体文件链接。                                                      |
| content        | str   | 原始内容。                                                      |

### 请求示例

**cURL**
```bash
curl -X GET https://api.reportify.cn/v1/docs/1046895208953942016/raw_content \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json"
```

**Python 示例**
```python
import requests

url = "https://api.reportify.cn/v1/docs/1046895208953942016/raw_content"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```
