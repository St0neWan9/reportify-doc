## 获取最新概念

**URL**: `/v1/concepts/latest`  
**方法**: `GET`  
**描述**: 获取最新概念。


### 请求参数
| 参数名 | 类型 | 描述          |
|--------|------|---------------|
| - | -  | - |


### 响应参数
| 参数名 | 类型 | 描述 |
|--------|------|------|
| concept_id | str | 概念唯一标识 |
| concept_code | str | 概念代码 |
| concept_name | str | 概念名称 |
| concept_desc | str | 概念描述 |
| concept_keywords | array[str] | 概念关键词列表 |
| concept_score | float | 概念标准分 |
| trending_score | float | 概念热度分 |
| scored_at | int | 概念计算时间戳 |
| published_at | int | 概念发布时间戳 |
| first_detected | boolean | 是否首次出现 |
| stocks | array | 相关股票列表 |
| &nbsp;&nbsp;market | str | 股票市场 |
| &nbsp;&nbsp;code | str | 股票代码 |
| &nbsp;&nbsp;name | str | 股票名称 |
| &nbsp;&nbsp;description | str | 股票描述 |
| &nbsp;&nbsp;relevance | float | 股票相关性分数 |
| docs | array | 相关文档列表 |
| &nbsp;&nbsp;doc_id | str | 文档唯一标识 |
| &nbsp;&nbsp;title | str | 文档标题 |
| &nbsp;&nbsp;url | str | 文档链接 |
| &nbsp;&nbsp;relevance | float | 文档相关性分数 |


### 请求示例

**cURL**
```bash
curl -X GET "https://api.reportify.cn/v1/concepts/latest" \
-H "Authorization: Bearer 447460****09c9" \
-H "Content-Type: application/json"
```

**Python 示例**
```python
import requests

url = "https://api.reportify.cn/v1/concepts/latest"
headers = {
    "Authorization": "Bearer 447460****09c9",
    "Content-Type": "application/json"
}

response = requests.GET(url, headers=headers)
print(response.json())
```
