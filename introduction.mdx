##  Reportify 是什么

Reportify 是一款基于 AI 大模型的投资研究深度内容问答引擎，可以方便的对沪深、港股、美股上市公司财报、电话会议、调研纪要、研报和新闻进行问答、总结、浏览、翻译。

使用方式：
浏览器直接访问 https://reportify.cn  

Reportify 亮点功能：
* 支持沪深、港股、美股全市场财报、电话会议、研报、新闻内容聚合
* 支持各类型多模态文档类型，可以对文本、音频、视频文件进行存储、分析
* 综合问答，可以针对全站内容和全网内容进行检索而且可以直接给出相应答案，并且带有引用出处
* 针对各种文档自动进行总结快速浏览文档核心内容和关键要点
* 每一个总结和关键要点都有引用出处并且对原文内容进行高亮，方便回溯原文
* 支持财报、电话会、新闻超长文档全文中英文对照翻译，以及选中文本定向翻译
* 支持历史问答会话保存，方便回溯和分享，构建投资研究知识库
* 支持用户自定义上传各种类型文档，方便构建私有知识库

## 文档概述
Reportify API 提供了一组功能强大的接口，旨在帮助用户通过 API 的方式访问 Reporitfy 的服务。通过这些 API，开发者可以实现文件的存储、搜索、问答等功能，并高效地获取和管理文档内容。本 API 文档详细介绍了各个接口的请求方法、参数要求、响应格式和示例，帮助开发者快速集成 Reportify 的功能。

## 示例应用场景
- **金融文档聚合**：将多个机构发布的报告集中存储和管理，便于跨公司、跨行业的分析。
- **智能内容检索和问答**：通过语义搜索和问答，快速定位特定领域或事件的关键信息，支持精准决策。
- **报告生成与总结**：利用文档内容与总结功能，为研究员、分析师提供简洁、直接的文档摘要，提升工作效率。

## 快速入门
在开始使用 API 前，请确保获取到一个有效的 API 密钥，并在请求的 `Authorization` 头中添加 `Bearer {API_KEY}` 以进行身份验证。
每个接口的请求示例和响应示例都采用 JSON 格式，开发者可以使用常见的 HTTP 客户端（如 `cURL`、`requests`）进行 API 测试与集成。
